Office Memory AI ("the App") is a Slack application built for the Slack Agent Builder Hackathon. By installing and using the App, you agree to the following terms.

## What the App Does
The App monitors messages in channels and direct messages where it has been added, detects team decisions, stores the reasoning behind those decisions, and answers user questions about past decisions using an AI language model.

## Data Usage
- The App processes Slack messages only in channels/DMs where it has been explicitly added by a workspace member.
- Message content identified as a "decision" is stored to enable future retrieval and question-answering.
- Message content is sent to a third-party AI language model (Alibaba Cloud DashScope / Qwen) at the time of each request, solely to generate responses. No data is used to train third-party models.
