# Privacy Policy — Office Memory AI

Last updated: July 2026

## What Data We Collect
Office Memory AI collects and stores:
- Messages identified as team "decisions" (the message text, timestamp, channel, and the Slack user ID of the sender)
- No other message content is permanently stored

## How Data Is Used
- Stored decision data is used solely to answer future questions asked within the same Slack workspace (e.g., "why did we choose X?").
- Message text is sent to a third-party AI provider (Alibaba Cloud DashScope / Qwen) at the time of each request, to generate a response. This data is not used to train third-party models and is not retained by the AI provider beyond the individual request.

## Data Storage
Data is stored on the App's hosting infrastructure (Railway) in a simple file-based store, scoped per Slack channel. Data is not shared with any party other than the AI provider used to generate responses.

## Data Retention
Decision data is retained for as long as the App remains installed in a workspace. Workspace admins may request deletion of all stored data at any time by contacting the developer.
